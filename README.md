## Features
- [x] Support for different Quiz APIs
    - [x] Categories
    - [x] Different difficulties
    - [x] API Stats
      
- [x] Local
    - [x] Creating and playing custom quizzes
    - [x] Storing API questions locally to play without Internet
    - [x] User stats
